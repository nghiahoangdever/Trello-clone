
# App quản lí công việc bằng mô hình kéo thả 

# Tính năng

- Tạo bảng (Board), danh sách (List) và thẻ (Card)
- Kéo thả thẻ giữa các danh sách (dnd-kit)
- Giao diện thân thiện, dễ làm quen với Material UI
- Tối ưu hiệu suất với Vite
- Hỗ trợ lưu trữ công việc

# Phiên bản
* nodejs = 22.15.0
* npm = v9.5.1
* yarn = v1.22.22
- Chi tiết các thư viện hơn xem trong 'package.json'

# Cách cài đặt:

- Tải repo dưới dạng zip hoặc clone về:
```
git clone https://github.com/nghiahoangdever/Trello-clone.git
```
- Setup task management app-web(Frontend):
```
yarn install
yarn dev
```
- Setup task management app-api(Backend):
```
yarn install
yarn dev
```
